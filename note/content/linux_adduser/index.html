Linux 新增sudo用户

  • 新增用户

使用root 用户登录

adduser jimbray

添加用户，然后根据提示输入密码

  • 加入 sudo 权限

修改 /etc/sudoers 文件权限

chmod 755 /etc/sudoers
  • 修改 sudoers 文件

复制一行

root    ALL=(ALL:ALL) ALL

最终结果为

root    ALL=(ALL:ALL) ALL
jimbray ALL=(ALL:ALL) ALL

完成。可以日常使用 jimbray 了。
